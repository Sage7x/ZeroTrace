# ZeroTrace
ZeroTrace — Detecting data exfiltration even when attackers leave no trace.

ZeroTrace is an open-source, Linux-first endpoint detection agent focused on spotting suspicious data collection, archiving, and outbound transfer behavior with a lightweight local runtime.

## Project Status

- Early-stage and documentation-first
- MVP target: local CLI-first Go agent for Linux
- No hosted service is required or assumed
